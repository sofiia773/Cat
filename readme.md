# 🐱 Nyan Cat Synth Project

# Description

Interactive audiovisual app built using Godot. Users can play synthesized sounds, trigger cat images, and activate a high-energy Nyan Cat disco mode.

# Features

* Real-time sound synthesis (sine wave)
* Play different notes using buttons
* Pitch control with slider
* Cat image interaction
* Meow sound effect (center button)
* Nyan Cat disco mode with music, rotation, and visual effects

# Controls

* Arrow buttons → play notes and show cats
* Slider → change pitch
* Center button → play meow sound
* Danger button → activate Nyan Cat mode

# How to Run

1. Open project in Godot
2. Run main scene
3. Use buttons to interact
4. Press Danger button for Nyan mode
